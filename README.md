# GestionArticle
Gestion d'Articles – Application CRUD  
Langage : Visual Basic .NET  
Framework ORM : Luna (génération automatique des requêtes SQL et des classes DAO)  
Base de données : SQL Server  

Fonctionnalités implémentées :
- Création, lecture, mise à jour et suppression d'articles
- Gestion des codes articles uniques
- Suivi automatique des dates de création et de modification
- Formulaire unifié pour l'ajout et la modification
- Recherche en temps réel (code ou désignation)
- Affichage sous forme de DataGridView avec formatage personnalisé
- Validation métier et gestion des erreurs (codes dupliqués, champs obligatoires, etc.)

Objectif : Démontrer une architecture propre avec séparation des couches (UI / Contrôleur / Modèle / DAO) et utilisation d'un générateur de code pour accélérer le développement.
